# java-cli-maven-spring-failsafe-pmd-jacoco-testng-test-hello-world

## Description
A POC for spring app using testng
framework with jacoco, pmd, and
failsafe plugins.

## Tech stack
- java
- maven
  - spring
  - testng
  - failsafe
  - jacoco
  - pmd

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- pmd report found at bin/target/site

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
